Setup VS Code for Jupyter Notebooks Using Anaconda

Step 1: Install Visual Studio Code

  1. Download and install Visual Studio Code from the official website: https://code.visualstudio.com/.
  2. Follow the installation instructions for your operating system (Windows, macOS, or Linux).

Step 2: Install Anaconda

  1. Download and install Anaconda from the official website: https://www.anaconda.com/products/distribution.
  2. Follow the installation instructions for your operating system.
  3. During the installation process, make sure to check the box that says "Add Anaconda to my PATH environment variable" if available (or set it up manually later).
  4. Verify the installation by opening a terminal (Command Prompt, PowerShell, or Terminal) and typing: 
    conda --version
    You should see the version of Conda you installed.

Step 3: Create a New Conda Environment (Optional but Recommended)

  1. Open a terminal.
  2. Create a new environment with a specific Python version (e.g., Python 3.8) by running: 
    conda create -n myenv python=3.8
    Replace myenv with your preferred environment name.
  3. Activate the new environment by running: 
    conda activate myenv

Step 4: Install Jupyter in the Conda Environment

  1. With the Conda environment activated, install Jupyter by running: 
    conda install jupyter

Step 5: Install the Python Extension in VS Code

  1. Open Visual Studio Code.
  2. Click on the Extensions icon in the Activity Bar on the side of the window (or press Ctrl+Shift+X).
  3. In the Extensions view, search for "Python".
  4. Click on the Python extension by Microsoft and install it.

Step 6: Install the Jupyter Extension in VS Code

  1. In the Extensions view, search for "Jupyter".
  2. Click on the Jupyter extension by Microsoft and install it.

Step 7: Configure Python Environment in VS Code

  1. Open VS Code and go to the Command Palette by pressing Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (macOS).
  2. Type "Python: Select Interpreter" and select it.
  3. Choose the Python interpreter from the Conda environment you created earlier. It will be listed as something like Python 3.x (Conda).

Step 8: Create or Open a Jupyter Notebook

  1. In VS Code, go to the Command Palette (Ctrl+Shift+P or Cmd+Shift+P).
  2. Type "Jupyter: Create New Blank Notebook" and select it to create a new Jupyter notebook.
    • Alternatively, you can open an existing .ipynb file by navigating to the file and opening it in VS Code.
  3. A new Jupyter notebook should open in a new tab. You can now add and run cells in the notebook.

Step 9: Running Jupyter Notebook Cells

  1. Add code or markdown to a cell.
  2. Run the cell by clicking the play button to the left of the cell or by selecting the cell and pressing Shift+Enter.

Step 10: Managing Extensions and Updates

  1. Keep your Jupyter and Python extensions up-to-date by periodically checking for updates in the Extensions view in VS Code.
  2. Regularly update your Conda packages by running: 
    conda update conda
conda update --all
